# 📚 Course Registry System

## 🚀 Project Overview

The **Course Registry System** is a full-stack web application developed using **Spring Boot, HTML, CSS, JavaScript, and MySQL**.
It allows users to register for courses and view available course details.

---

## 🛠️ Tech Stack

* Backend: Spring Boot (Java)
* Frontend: HTML, CSS, JavaScript
* Database: MySQL
* Tools: IntelliJ IDEA, VS Code, Git, GitHub

---

## ✨ Features

* Register new courses
* View all available courses
* Store data in database
* Simple and user-friendly interface

## ⚙️ Setup Instructions

### 1. Clone Repository

```
git clone https://github.com/your-username/Course-Registry-System.git
```

### 2. Configure Backend

Update `application.properties`:

```
spring.datasource.url=jdbc:mysql://localhost:3306/course_db
spring.datasource.username=your_username
spring.datasource.password=your_password
```

### 3. Run Backend

* Open project in IntelliJ
* Run Spring Boot application

### 4. Run Frontend

* Open HTML file in browser
  or
* Use Live Server in VS Code
